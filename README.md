# Dhanya-Krishna-portfolio.github.io
This portfolio website is a terminal-inspired personal dashboard designed to showcase my skills, projects, and interests in a unique and interactive format. The website follows a dark-themed UI with neon green accents, mimicking a command-line interface to reflect my interest in competitive programming and problem solving.

The site includes multiple sections such as Home, About, Projects, and Contact, accessible through a navigation bar that enables smooth scrolling across the page. The Home section introduces me and highlights my passion for algorithms and coding, while the About section provides details about my education, interests, and technical skills using visual skill bars.

The Projects section presents a structured view of my work in a problem-solving format, reinforcing the competitive programming theme. Additionally, the Contact section allows users to interact through a simple input form.

The website is built using HTML and CSS, with features like responsive layout, hover effects, animations (such as a blinking cursor), and styled components to enhance user experience. Overall, the design emphasizes both functionality and creativity, creating a visually engaging and professional portfolio.
